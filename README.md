# 📖 About Repository

This repository contains my **Database Management System (DBMS)** practice using **PostgreSQL**.

The goal is to build strong SQL problem-solving skills by solving questions of varying difficulty levels while learning the theoretical concepts taught in my university DBMS course.

Problems are categorized into:

- 🟢 Easy
- 🟡 Medium
- 🔴 Hard

The repository is continuously updated throughout the semester with new SQL queries, assignments, practicals, and advanced database concepts.
